Algorythmes

Comparateurs de variables :

  • == : égal à
  • === : égal et de même type
  • != : inégal
  • !== : inégal et/ou d'un type différent
  • && : et
  • || : ou
  • ! : not

Les conditions s'écrivent if(conditions)

Les conséquences s'écrivent else{conséquences}

Fonctions

Console.log est une fonction qui sert à afficher dans l'inspecteur

Return permet à envoyer les conséquences des algorythmes sur le site, d'être traité en dehors de l'inspecteur uniquement
