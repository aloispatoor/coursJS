Informations Générales

  • JavaScript peut s'écrire dans le fichier html via les balises "script"
  • Lier le fichier js en fin de page HTML avec la balise script + attribut src avec le chemin du fichier .js. Facultatif : attribut "type"="text/javascript"
  • Les commentaires en JS sont fait de 2 manières : // ou /* */
  • Extensions : Javascript (ES6) code snippets et Prettier Code Formatter
  • Créer une alerte : alert("message");
  • Le js est visible dans l'inpecteur dans l'onglet "Console"
  • Le "console.log" permet d'envoyer un message et également d'afficher le contenu des variables à la console de l'inspecteur.
  • Le JS a une préférence pour le Camelcase
  • Une variable est une boîte où stocker des informations. Elle peut contenir du texte (string), des chiffres, des booleans, des objets (objects) et des tableaux (array)
  • Si variable vide : Undefined
  • Si variable = 0 : NULL
  • Les cotes "" indique une string, du texte
  • Les chiffres et les booleans n'ont pas besoin de syntaxe particulière
  • Les booleans : true / false
  • Pour déclarer une variable, on la précéde de let ou de var (plus ancien)
  • Si précédée de const, la variable est constante, elle ne peut pas changer
  • Concaténation : Pour afficher une suite de variable : console.log(variable1 + variable2);
  • Autre exemple de concaténation : console.log(`string ${variable}`)
